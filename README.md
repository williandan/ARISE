## Sobre

Um portfólio de desenvolvedor onde **o visitante é o jogador**. Ao navegar pelo site, você cumpre missões, ganha XP e sobe de rank (E → S) — como o Sung Jin Woo — desbloqueando os projetos (Gates) de rank mais alto conforme evolui. Feito com Next.js, React e Node, bilíngue (PT/EN), com estética das "janelas azuis do Sistema".

> A camada de jogo é opcional: todo conteúdo essencial (projetos, contato, currículo) fica acessível mesmo pra quem não quer jogar.

## Stack

`Next.js` · `React` · `TypeScript` · `Node.js` · `Tailwind CSS` · `Framer Motion` · `GSAP` · `Zustand` · `Howler.js` · `Vercel`
